🎨 AI Photo Editor - Chrome Extension Icons

这个页面会生成Chrome扩展所需的三种尺寸图标（16x16, 48x48, 128x128）

📋 使用说明：

  1. 点击"生成图标"按钮创建图标
  2. 右键点击每个图标选择"图片另存为"
  3. 保存为对应的文件名：icon16.png, icon48.png, icon128.png
  4. 将文件放到 chrome-extension/assets/icons/ 目录下
16x16 (工具栏)
48x48 (扩展管理)
128x128 (Chrome商店)

🎯 图标设计说明：

  • 主色调：橙黄渐变（与网站主题一致）
  • 图标元素：AI + 图像符号
  • 风格：现代化、简洁、易识别
  • 背景：圆角矩形，适合各种背景
